# Pizza Parlor

#### By Ebru Rice

#### Pizza Parlor is a website to order pizza.

## Technologies Used

* HTML
* CSS
* JavaScript

## Description

A website for a pizza company where a user can choose one or more individual toppings and a size to order a pizza and see the final cost.
